# 6. Project name selection

Date: 2026-06-27
## Status
Accepted

---

## Context

We needed a memorable, easy-to-type, relevant name with clean namespaces (as the Rust engine publishes to crates.io).

---

## Decisions

### Selected Name: jsonette
**Decision:** Use the name **jsonette** for the project, crate, and repository.

**Rationale:**
- Verified free on crates.io, npm, GitHub organization, and Homebrew cask.
- The "-ette" diminutive signals "lightweight," matching the product thesis.

---

## Consequences

- No namespace workarounds or conflicts.
- **Alternatives Considered:**
  - *jsonit* (taken on crates.io, npm, and GitHub).
  - *jsonix* (free on crates.io, but npm was held by an existing XML↔JSON library, and GitHub organization was taken).
