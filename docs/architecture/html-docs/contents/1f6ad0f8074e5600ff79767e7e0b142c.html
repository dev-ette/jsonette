CompletionItem

UMLDataType
jsonette :: model :: types :: CompletionItem

Description

One autocomplete suggestion for the query input. When the user has typed $.users[0]. the engine walks the live JsonNode and returns [{key: "name", path: "$.users[0].name"}, ...].
