FormatOptions

UMLDataType
jsonette :: model :: types :: FormatOptions

Description

Parameters for pretty-printing. Stateless configuration the shell passes in; the engine doesn't keep state.

Attributes

Visibility Name Type Description
public use_tabs bool

Whether to use tab characters (\t) for indentation
public indent u8

The number of spaces to use for indentation.
public line_ending LineEnding

The line ending character sequence to use.
public folding_style FoldingStyle

The wrapping/folding strategy for objects and arrays.
public sort_keys bool

Whether to sort objects keys alphabetically.
public space_after_colon bool

Whether to insert a space after colons (e.g. {"key": "value"} vs {"key":"value"}).
public space_after_comma bool

Whether to insert a space after commas (e.g. [1, 2] vs [1,2]).
