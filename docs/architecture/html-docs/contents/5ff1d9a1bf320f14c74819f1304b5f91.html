Parser

UMLClass
jsonette :: model :: Modules :: Parser :: Parser

Description

A stateful recursive-descent parser for strict JSON documents. Keeps track of byte offset locations to generate AST nodes with accurate Span info.

Visibility Name Type Description
private input u8 [*]

The input bytes slice of the JSON document.
private input_raw String

The original input string slice for number parsing and error reporting.
private cursor usize

The current byte offset cursor in the input.
