Diagnostic

UMLDataType
jsonette :: model :: types :: Diagnostic

Description

A single problem found during parsing. One or many of these form the engine's complete error report. The shell takes the span, maps it to screen coordinates, and draws a red squiggle.

Attributes

Visibility Name Type Description
public span Span

The span of the input text where this diagnostic applies.
public message String

The diagnostic message explaining the error or warning.
