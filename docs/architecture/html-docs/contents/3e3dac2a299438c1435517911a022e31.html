JsonNode

UMLClass
jsonette :: model :: types :: JsonNode :: JsonNode

Description

The parsed JSON tree in memory. A recursive type: a node is either a string, number, bool, null, an array of nodes, or an object (map of key → node). This is the in-memory representation that every other function in the engine operates on.
