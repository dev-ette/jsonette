Span

UMLDataType
jsonette :: model :: types :: Span

Description

A byte-offset range inside the source text string. Not a line/column yet — that conversion is the shell's job. Purpose: the engine needs a universal, UI-agnostic way to say "the error is here" without knowing how the shell will render it.
